# ffg: fish-fzf-git
fish git key bindings with fzf

# Install 
```shell
fisher add ikedaosushi/ffg
```

# Usage

| Keybindings | Remarks                          |
| --------------- | ----------------------------------------------- |
| Ctrl-c          | git checkout local branch                                   |
| Ctrl-o          | git checkout local commit                |
| Alt-e           | git checkout -b remote branch |
